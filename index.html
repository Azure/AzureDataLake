Azure Data Lake

http://aka.ms/AzureDataLake

Overview

Azure Data Lake comprises three cloud-based services (HDInsight, Data Lake Analytics, & Data Lake Store) that make it easy to handle store an analyze any kind of data in Azure.

To get a quick overview of the services see the Azure Data Lake page on Azure.com
